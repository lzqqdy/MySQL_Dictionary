# MySQL_Dictionary
php编写的MySQL数据字典
* php > 7.0
* mysql >5.4
> 在本地项目跟目录下clone后
> 访问方式：http://127.0.0.1/MySQL_Dictionary/mysql.php?name=fastadmin&pwd=root
###参数
```
name：数据库名称
pwd：访问密码
```
